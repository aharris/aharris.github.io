This project, for the Liberty Global company Telenet (code name: DAWN), was an extremly unique undertaking. DAWN is a single page app that drives the UI and UX for a new cable set top box being rolled out in multiple countries. My responsibilities included creating and styling all views utilizing dust.js templates and the Compass CSS authoring framework as well as implementing functionality with strict vanilla JavaScript. I used CSS3 animations to achieve smooth animation performance on an embedded device running QtWebKit. I also was in charge of managing i18n dictionaries and app wide translatable tags for multi-lingual support. I gained valuable on-site experience while working directly with clients in Mechelen, Belgium to rapidly implement changes and perfect UI animations.
